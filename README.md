# Pronto!

Pronto! is a reservation booking system built to showcase a fullstack portfolio project.

## Stack

- Frontend: React + TypeScript
- Backend: Node.js + Express
- Database: MongoDB or MySQL
- Authentication: JWT

## Objective

Build a fullstack application with:

- User registration and login
- CRUD for reservable resources
- Booking with availability validation
- User dashboard and admin panel
- Production-ready deployment

## Project structure

- `/client`: React application
- `/server`: Express backend

> All code files, routes, component names, and identifiers should use English. The website UI content may remain Spanish.

## Getting started

1. Clone the repository
2. Install dependencies in `/client` and `/server`
3. Configure environment variables from `.env.example`
4. Run the frontend and backend
